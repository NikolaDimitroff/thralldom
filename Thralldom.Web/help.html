Our help section is currently limited due to small number of features we presently have.
Nevertheless, here are a few notes you might find helpful.

Running the game

I don't see the download button, how am I supposed to play?
The game runs entirely in the browser, you need not download any executable files. This allows us to support virtually every platform we are interested in and update the game without you even noticing an update is going on.
What do I need to play? Any hardware / software requirements?
We haven't gathered enough test data to provide you with precise system requirements but you should be fine with every machine running a moderately powerful CPU / GPU. We have software requirements though. You need to make sure that you have your latest video card drivers installed. Also, we currently only support Internet Explorer 11, Google Chrome and Mozilla Firefox (make sure they're updated!). We make no plans for supporting neither Opera nor Safari at the moment.
How do I play?
Well, you click on the "Play" button on the home page. I assume you are too lazy to go back there, so here's a direct link

We have a large amount of assets you need to download (models, textures, etc.). They take some time to download (as they would've taken with any other desktop game) but worry not - we're caching all those files so the next you start the game, you can even run it without internet access!
Do I need internet access to play?
Only the first time the game is loaded (and whenever you'd like to update). Take a look at the previous question.

Playing

What do I do after I begin the game?
Follow the mission objectives in the top left angle of the screen.
How do I move?
We are using standard action game keybindings. Use WASD to move, Space to jump and your LMB (Left Mouse Button) to attack (if you have a weapon equipped).
I finished my mission, now what?
Currently, we only have a single playable mission so after you finish it, there's not much to do but to wait until our next release. We also would really appreaciate if you contact us for feedback.
