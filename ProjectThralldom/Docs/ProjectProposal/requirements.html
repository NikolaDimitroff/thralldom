Thralldom

Software instructions

Below are the requirements for running Thralldom:

  • Up-to-date video card drivers
  • One of the following browsers: Internet Explroer 11, Chrome 34, Firefox 28 or newer
  • Enabled Javascript in the browser
  • Active internet connection for loading the game the first time and using the cloud service
  • In order to run the game from the provided archive, one must extract the archive and use a web server (such as IIS) to serve the files (unlike going to http://thralldom.net)
